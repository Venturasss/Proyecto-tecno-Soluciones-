# tecnoSoluciones S.A. - Gestión Empresarial 🚀

Sistema web moderno desarrollado en PHP con arquitectura MVC para centralizar la administración empresarial, el control de accesos y la automatización de documentos contables.

## 🌟 Características Clave
- **Autenticación Segura:** Panel de login optimizado con validación de credenciales.
- **Módulo de Facturación:** Procesamiento automatizado de montos y cálculo exacto de facturas con el 18% de IGV.
- **Administración Centralizada:** Control e historial de clientes, estados de proyectos y presupuestos.
- **Reportes al Instante:** Módulo integrado para la exportación de reportes limpios y profesionales.
- **Diseño Premium:** Interfaz responsiva en modo oscuro con una estética minimalista y limpia.

## 🛠️ Tecnologías Utilizadas
- **Backend:** PHP (Arquitectura limpia basada en controladores y enrutamiento dinámico).
- **Frontend:** HTML5, CSS3, JavaScript, componentes responsivos modernos.
- **Base de Datos:** Estructura relacional optimizada.
